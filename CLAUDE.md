# CLAUDE.md — partner-finder-fermi

婚活フェルミ推定Webアプリ。日本の官公庁統計データをもとに、設定した条件に合う未婚者が何人いるかを推計・表示する。

## コンセプト

「自分の条件だと該当する未婚者が実際何人いるのか」を可視化することで、婚活条件を冷静に見直すきっかけを提供する。元ネタ: https://x.com/haru_tachibana8/status/2044252171961807201

## 構成

- **単一HTMLファイル** (`index.html`) — サーバー不要、ブラウザで開くだけ
- 外部API・バックエンド不要
- データはすべてJavaScript内にハードコード（出典明記）

## データソース（必ずアプリ内に明記すること）

| データ | 出典 |
|---|---|
| 未婚人口・年齢別 | 総務省 令和2年(2020年)国勢調査 |
| 年収分布（男女別） | 国税庁 令和6年分(2024年)民間給与実態統計調査 |
| 身長分布 | 厚生労働省 令和4年(2022年)国民健康・栄養調査 |
| 喫煙率 | 厚生労働省 令和4年(2022年)国民健康・栄養調査 |
| 学歴（大卒以上割合） | 総務省 令和2年(2020年)国勢調査 |
| 都道府県別人口 | 総務省 令和2年(2020年)国勢調査 |
